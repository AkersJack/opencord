Run
---

.. code-block:: text

    $ flask --app server init-db
    $ flask --app server run --debug

OR on Windows (in powershell)

.. code-block:: text
    $ ./initdb.ps1
    $ ./start.ps1

On Windows (in CMD)
.. code-block:: text
    $ ./initdb.cmd
    $ ./start.cmd


Open http://127.0.0.1:5000 in a browser.
